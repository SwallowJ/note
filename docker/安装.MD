# 安装

## 安装 docker

详见: https://mirror.tuna.tsinghua.edu.cn/help/docker-ce/

## 添加到 docker 用户组

```bash

# 查看docker组
cat  cat /etc/group |grep docker

# 添加用户组
sudo groupadd docker

# 添加用户到docker组
sudo gpasswd -a ${USER} docker

# 重启docker
sudo service docker restart

#切换当前会话到新 group
newgrp - docker
```
