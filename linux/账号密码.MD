# 账号管理

## 设置 ROOT 密码

```bash
sudo passwd
```

## 新增用户

```bash
adduser xxx
```

## 切换用户

```bash
su
```

## 删除用户

```bash
deluser
```

## 免密登录

```bash
ssh-keygen -t rsa #这里用-t指定加密算法

#不存在authorized_keys
mv id_rsa.pub ~/.ssh/authorized_keys

#存在authorized_keys
cat id_rsa.pub >> ~/.ssh/authorized_keys
```
